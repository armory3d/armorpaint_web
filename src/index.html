3D Painting Software

ArmorPaint is a stand-alone software designed for physically-based texture painting. Drag & drop your 3D models and start painting. Receive instant visual feedback in the viewport as you paint.

Node Based

Work fast with the convenience of nodes. Paint with fully procedural materials. Build fill layers with material nodes. Use brush nodes to create patterns and procedural brushes.

Edge Wear

GPU Accelerated

ArmorPaint is designed from scratch to run completely on the GPU. This results in a smooth 4K painting experience on a medium-power integrated hardware. Up to 16K texture painting is seamless using a high-end graphics card.

Ray Tracing

Ray-traced baking, painting effects and viewport rendering running on Direct3D12, Vulkan and Metal is ready. ArmorPaint is powered by an in-house 3D rendering engine to achieve the best painting experience.

Subsurface Scattering

Live Link

Integrate ArmorPaint into existing 3D software using the live-link plugins. Blender, Unreal Engine and Unity plugins are available for download.

Baking

Bake texture maps for high-poly models instantly on your GPU. Hardware accelerated ray-traced baking with Direct3D12, Vulkan and Metal support is available.

Decal Layers

Path Traced Viewport

Paint models in a viewport with true light simulation. Path-traced viewport rendering with Direct3D12, Vulkan and Metal is available.

Plugins

Enhance any part of the software through plugins with super powers. Integrate new node systems or build custom material nodes.

Texture Creation

Create seamless textures from text prompts or use existing photos.

Edit & Upscale

Edit textures with text prompts. Create image variations. Use the inpaint mask to fill in details or remove unwanted parts. Upscale the images up to 4x.

Node Workflow

To Material

Extract base color, height, normal map, occlusion and roughness textures from a photo.

Paint & Export

Use the generated materials to paint your models in ArmorPaint or export the PBR textures to disk.

All neural node processing in ArmorPaint is done locally on user device. Once AI models are downloaded, internet connection is not required. A minimum of 6GB of video memory is recommended. ArmorPaint uses models published under open source licenses.

Sculpting

In development - sculpt 3D models. Use non-destructive layer workflow to refine the mesh with details. All tools from paint workflow available for sculpting.

Node Based

Sculpt with procedural materials and brushes. Build fill layers and masks with material nodes. Use brush nodes to create sculpting patterns.

Runs Everywhere

ArmorPaint runs on Windows, Linux, macOS, iOS and Android. On desktop, ArmorPaint is a portable application with no installation, near-instant start up and download size of less than 10MB.
